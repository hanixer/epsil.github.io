Nyttige snarveier i Visual Studio

2016-05-25

Snarvei Funksjon
Ctrl + K, Ctrl + C Kommenter
Ctrl + Alt + 7 Kommenter med ReSharper*
Ctrl + ⇧ + 7 Blokk-kommenter med ReSharper*
Ctrl + K, Ctrl + U Avkommenter
Ctrl + K, Ctrl + F Indenter
Ctrl + K, Ctrl + D Indenter alt
Ctrl + . Refaktorer
Ctrl + L Slett linje
Alt + ↵ Refaktorer med ReSharper*
Ctrl + R, Ctrl + R Omdøp variabel*
Ctrl + - Gå til forrige posisjon
Esc Avbryt IntelliSense
Ctrl + Alt + M, Ctrl + H Unity API Reference**

*Krever ReSharper.
**Krever Visual Studio 2015 Tools for Unity.
